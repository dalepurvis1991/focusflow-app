📦 INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Navigate to project:
   $ cd focusflow-app

2. Install dependencies:
   $ npm install

3. Start development server:
   $ npm run dev

4. Open browser:
   → http://localhost:3000

5. First time? Complete onboarding:
   ✓ Enter name & age
   ✓ Set daily routine
   ✓ Optional: ADHD medication info
   ✓ Enable notifications
   ✓ Start using!

   • Time-aware greeting
   • Today's schedule summary
   • Current streak tracker
   • Quick action buttons
   • Gentle nudge reminders

🤖 Assistant
   • Chat with AI coach
   • Mock responses for common concerns
   • Voice input button (placeholder)
   • Persistent message history

📅 Calendar
   • Week view with navigation
   • Color-coded events
   • Sample events included:
     - Monday: Doctor appointment, team meeting
     - Tuesday: Project deadline
     - Wednesday: Coffee with Alex
   • Add event modal

⚙️ Profile
   • View your info
   • Manage preferences
   • Reminder frequency control
   • Focus session duration
   • Break reminders toggle
   • Sign out

   • User profile & settings
   • Calendar events
   • Chat history
   • Onboarding completion

Reset anytime: Profile → "Reset Onboarding"
